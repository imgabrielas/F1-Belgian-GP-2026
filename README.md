# F1 Race Winner Prediction from Qualifying Data

Predicting the winner of the Belgian Grand Prix using qualifying session data.

## Status

Work in progress — not finished yet.

## Goal

Build a machine learning model that predicts the race winner (not the full podium) based on qualifying results. Data will be collected via the [FastF1](https://github.com/theOehrly/Fast-F1) API, with **qualifying lap times as the main feature**.

## Tech Stack

- Python 3.13
- FastF1
- pandas, numpy
- seaborn, matplotlib
- scikit-learn
- Jupyter Notebook
